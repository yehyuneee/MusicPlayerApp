## MusicPlayerApp (음악 재생 플레이어)

* 앱 구성 화면은 다음과 같습니다.
- 스플래시 스크린
- 음악 재생 화면
1. 재생 중인 음악 정보(제목, 가수, 앨범 커버 이미지, 앨범명)
2. 현재 재생 중인 부분의 가사 하이라이팅
3. Seekbar
4. play/Stop 버튼
- 전체 가사 보기 화면
1. 특정 가사로 이동할 수 있는 토글 버튼
2. 전체 가사 화면 닫기 버튼
3. Seekbar
4. Play/Stop 버튼
          
*** 현재 각 화면 UI 구성 및 데이터 모델링 구현 중
